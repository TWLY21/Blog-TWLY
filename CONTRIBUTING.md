# Contributing

Thanks for your interest in improving this project.

## Development Setup

1. Fork the repository.
2. Clone your fork.
3. Install dependencies:

```bash
npm install
```

4. Start dev server:

```bash
npm run dev
```

## Pull Request Guidelines

- Keep changes focused and scoped.
- Follow existing code style and folder structure.
- Ensure TypeScript checks pass:

```bash
npx tsc --noEmit
```

- Include a clear PR description with:
  - what changed
  - why it changed
  - screenshots for UI updates

## Commit Guidelines

- Use concise, descriptive commit messages.
- Group related changes in one commit where possible.
